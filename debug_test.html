Troubleshooting

If main app shows blank page:

  • Hard refresh: Cmd+Shift+R or Ctrl+Shift+R
  • Clear browser cache and try again
  • Check browser console (F12) for errors
  • Try incognito/private browsing mode
  • Restart the server: ./start.sh
